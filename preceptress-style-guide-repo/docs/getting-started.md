# Getting Started

## 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/preceptress-style-guide.git
cd preceptress-style-guide
```

## 2. Open the guide

Because the current guide is fully self-contained, you can open it directly in a browser:

```bash
open index.html
```

On Linux:

```bash
xdg-open index.html
```

## 3. Make your first edits

The easiest first edits are:

- update the brand name in the `<title>` and hero section
- update color tokens in `:root`
- change button labels and component names
- add screenshots or product examples for your current site

## 4. Preview often

Refresh your browser after each small change. This is the fastest way to protect layout quality.

## 5. Commit and push

```bash
git add .
git commit -m "Initial public style guide"
git push origin main
```
