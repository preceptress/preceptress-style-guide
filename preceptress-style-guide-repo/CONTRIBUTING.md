# Contributing

Thank you for helping improve the Preceptress Health style guide.

## Goals

This repository should remain:

- visually consistent
- easy to fork and reuse
- readable by non-designers
- credible for researchers, funders, and technical collaborators

## Before you change anything

Please keep the design language aligned with the current direction:

- dark scientific interface
- clear visual hierarchy
- premium but restrained polish
- accessible contrast and readable sizing
- reusable classes over one-off styling

## Recommended workflow

1. Create a branch.
2. Make focused changes.
3. Test the page in a desktop browser and on a narrow mobile viewport.
4. Open a pull request with screenshots if the visual output changed.

## Commit style

Use clear commit messages such as:

- `Refine card spacing in hero panel`
- `Add documentation for GitHub Pages deployment`
- `Update color token naming for consistency`

## Pull request checklist

- [ ] The page still loads as a standalone HTML file.
- [ ] New styles use existing tokens where possible.
- [ ] Text remains readable on dark backgrounds.
- [ ] Mobile layout is not broken.
- [ ] Documentation was updated if behavior changed.
